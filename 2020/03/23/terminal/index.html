Terminal 使用小技巧

Posted on 2020-03-23

bash使用技巧

bash是一款基础shell。
配置文件名：.bash_profile
改变Terminal行首显示：在配置文件中添加
export PS1="xxx"
xxx中填写想要的行首(prompt)
prompt variables 包括：
\A: 以24小时制，参考时间
\w: 显示current working directory
