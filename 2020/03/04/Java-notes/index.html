Java的对象和类

Java 包括以下几项基本概念：

  1. 多态性
  2. 继承性
  3. 封装性
  4. 抽象性
  5. 类
  6. 对象
  7. 实例
  8. 消息解析
    所谓类和对象，就像是蓝图和一个具体的物品
    类是独立的对象创造出来的蓝图
    每个类都包含了属性（attribute）和方法（methods）（也就是函数）。
    举例：一个汽车类，属性有品牌，排量，发动机型号，车辆尺寸等，方法可以有汽车踩油门，踩刹车，开锁，关锁，加油等。

    Java Scanner类的使用

    import java.util.Scanner; //被打包在java.util中
    Scanner scan = new Scanner(System.in); //建立一个对象scan
    int a = scan.nextInt();
    double b = scan.nextDouble();
    String c = scan.next(); //输入下一个字符串
    String c = scan.nextLine(); //输入下一排字符串
    Char d = scan.next().charAt(); //输入一个char字符
    boolean e = scan.hasNext(); //判断是否有输入
    boolean f = scan.hasNextInt(); //判断是否有整型数字输入
